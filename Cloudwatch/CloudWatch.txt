Amazon CloudWatch is a monitoring service for Amazon Web Services cloud resources and the applications you run on Amazon Web Services.CloudWatch collects and tracks metrics, collects and monitors log files, and sets alarms to alert you on certain conditions.

Problems solved by cloudwatch:-

- Resource utilization
- Proactive Monitoring
- Troubleshooting
- Scalability

CW performs:-

- Monitoring
- Real life metrics
- Alarms
- Log insights
- Custom metrics
- Cost optimization
- Autoscaling


Project: We'll up CloudWatch alarms for critical metrics of an application, define appropriate threshold 
conditions, and configure notification actions.
